HTML Links

A link is specified using HTML tag . This tag is called anchor tag and anything between the opening tag and the closing tag becomes part of the link and a user can click that part to reach to the linked document.



The target Attribute This attribute is used to specify the location where linked document is opened. Following are possible options:



Use of Base Path When you link HTML documents related to the same website, it is not required to give a complete URL for every link. Use tag in your HTML document header. This tag is used to give a base path for all the links. So your browser will concatenate given relative path to this base path and will make a complete URL.
